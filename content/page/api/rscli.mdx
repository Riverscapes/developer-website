The Riverscapes Command Line Interface (RSCLI) is a small, lightlight application for interacting with the [Riverscapes Data Exchange](https://data.riverscapes.net) and the associated [API](./api). RSCLI simplifies the process of uploading and downloading data from the Riverscapes Data Exchange. It handles authenticating your user account and validating data uploads.

# Installing the RSCLI

Open a command line prompt (Terminal on Mac, Command Prompt on Windows) and run the following command:

```bash
npm install -g @riverscapes/cli
```

# Using the RSCLI

The RSCLI is a command line application. To use it, open a command line prompt and run the `rscli` command. The application has excellent POSEX help for all the available commands and options. To see the help, run the following command:

```bash
rscli --help
```

For help uploading data, run the following command:

```bash
rscli upload --help
```

For help downloading data, run the following command:

```bash
rscli download --help
```
